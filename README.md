# Portfolio
Portfolio professionnel présentant mes projets.
## Projets
- Voyage
- Médiathèque
- Memory
## Technologies
- Node.js / Express
- Tailwind CSS v4
- JavaScript
## Installation
```bash
npm install
npm run build:css

npm start
```
## Scripts
- `npm start` : Serveur
- `npm run build:css` : Compile CSS
- `npm run watch:css` : Watch mode
